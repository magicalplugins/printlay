A **cut contour** is the vector outline a cutter follows around a design. By convention it's assigned to a [spot colour](/glossary/spot-colour) named exactly **`CutContour`**, which print-and-cut [RIP software](/glossary/rip-software) such as Roland VersaWorks recognises and sends to the cutter instead of printing.

The cut path is normally set to overprint so it doesn't knock out the artwork beneath. See [sticker cut lines](/guides/sticker-cut-lines).
