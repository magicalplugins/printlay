A **cut line** is a vector path that tells a cutter or plotter where to cut around a printed design. It isn't printed as ink — it's an instruction, usually marked as a [spot colour](/glossary/spot-colour) named **CutContour** that the RIP routes to the cutter.

A sticker file therefore has two parts: the printed artwork (with [bleed](/glossary/bleed)) and the cut path. See [sticker cut lines](/guides/sticker-cut-lines).
