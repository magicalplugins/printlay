**PNG** (Portable Network Graphics) is a **lossless raster image format** that supports a **transparent background**. Because transparency tells a RIP exactly where to print — and where to generate the white underbase — a high-resolution transparent PNG (300 DPI at print size) is the common default for [DTF](/glossary/dtf) and sticker artwork.

Unlike JPEG, PNG doesn't lose quality to compression or flatten transparency. See [best file format for DTF](/blog/best-file-format-for-dtf) for when to use PNG, TIFF or PDF.
