**RIP** stands for **Raster Image Processor**. RIP software converts your design file into the data a printer outputs — handling colour management, resolution, white-ink layers and, in print-and-cut workflows, sending [cut contours](/glossary/cut-contour) to the cutter.

Examples include Roland VersaWorks and other print/cut RIPs. The RIP is what recognises a `CutContour` [spot colour](/glossary/spot-colour) as a cut path rather than ink.
