**Vector** artwork is built from **mathematical paths** (points, lines and curves) rather than pixels, so it scales to **any size without losing quality**. That makes it ideal for logos, text, and anything a cutter follows — [cut lines](/glossary/cut-line), [HTV](/glossary/htv) designs and [cut contours](/glossary/cut-contour).

The opposite is a [raster](/glossary/raster) image, which is made of pixels and degrades when enlarged. Cutters need vector paths to know where to cut; printers can use either, but vector keeps edges crisp at any scale.
