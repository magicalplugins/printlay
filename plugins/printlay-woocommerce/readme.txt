Integrate the PrintLay custom sticker designer into your WooCommerce product pages.

== Description ==

PrintLay WooCommerce adds a fully-featured sticker designer to your WooCommerce store. Customers can upload artwork, choose sizes, materials, and finishes, see live pricing, and add their custom sticker to cart — all without leaving your site.

**Features:**

* Full-screen overlay designer (matches PrintLay's own preview exactly)
* Live pricing calculated by PrintLay's pricing engine
* Supports cut-out and canvas (shaped) designer modes
* Design thumbnail shown in cart and orders
* "From £X.XX" price display for SEO/Google Shopping
* Tamper-proof pricing via signed quote tokens
* Each design gets its own cart line with full options summary

== Installation ==

1. Upload the `printlay-woocommerce` folder to `/wp-content/plugins/`
2. Activate the plugin via the Plugins menu
3. Go to PrintLay > Settings and enter your API key
4. On any WooCommerce product, enable PrintLay Designer and select a linked product

== Setup ==

1. Create a PrintLay account at https://printlay.co.uk
2. Set up your pricing profiles and products in PrintLay
3. Generate an API key in PrintLay > Widget > API Keys
4. Add your store domain to PrintLay > Widget > Settings > Allowed Origins
5. Install this plugin and enter your API key
6. Link WooCommerce products to PrintLay products
